# 🛒 Blinkit Sales Analysis Dashboard (Excel)

## 📊 Project Overview

This project focuses on analyzing sales performance, customer satisfaction, and inventory distribution for **Blinkit**, a quick-commerce grocery delivery platform owned by Zomato.

The objective of this analysis is to identify key business insights that help understand product demand, outlet performance, and customer preferences.

Using **Microsoft Excel**, a fully interactive dashboard was developed to visualize business KPIs and sales trends, enabling users to explore the data dynamically through filters and slicers.

---

## 🎯 Business Problem

Blinkit needs a clear understanding of its **sales performance, product demand, and outlet distribution** across different cities and store formats.

The goal of this project is to:

* Analyze overall sales performance
* Understand customer ratings and satisfaction
* Evaluate sales distribution across outlet types, sizes, and locations
* Identify top-performing product categories

The final solution is an **interactive Excel dashboard** that provides both **high-level business summaries and detailed insights**.

## 🧹 Data Cleaning & Preparation

Before building the dashboard, the dataset required cleaning and preparation.

Key steps included:

**Standardizing Values**

The *Item Fat Content* column had inconsistent values. Using **Find & Replace**, the following corrections were made:

* "LF" and "low fat" → **Low Fat**
* "R" and "reg" → **Regular**

**Table Conversion**

The dataset was converted into an **Excel Table (Ctrl + T)** to ensure Pivot Tables update dynamically.

**Additional Column**

A **Serial Number column** was created to accurately count the total number of items sold.

---

## ⚡ Dashboard Interactivity

The dashboard is fully interactive using **Excel Slicers**, allowing users to filter the data dynamically.

Available filters include:

* Outlet Size (Small / Medium / High)
* Outlet Location Type (Tier 1 / Tier 2 / Tier 3)
* Item Type

All slicers are connected to the Pivot Tables using **Report Connections**, enabling all charts to update simultaneously.

Navigation buttons were also added to switch between:

* Dashboard
* Pivot Tables Sheet
* Raw Data Sheet

---

## 🔍 Key Insights

From the analysis, several important insights were discovered:

* **Snack foods and fruits are among the highest-selling product categories.**
* **Medium-sized outlets generate the highest sales.**
* **Low-fat products contribute significantly to total sales.**
* **Tier 2 cities show strong demand for quick commerce services.**
* Older outlets tend to generate more consistent sales.

---

## 💡 Business Recommendations

Based on the analysis:

* Increase inventory for high-demand product categories like snacks and fruits.
* Focus expansion strategies on **Tier 2 cities**.
* Optimize medium-sized outlets for better operational efficiency.
* Promote healthier product options such as **low-fat products**.
