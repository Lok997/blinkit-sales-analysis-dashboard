# 🛒 Blinkit Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

This project analyzes a **Blinkit-style retail sales dataset** to uncover insights about product demand, outlet performance, and customer preferences in the quick-commerce industry.

Using **Power BI**, an interactive dashboard was built to visualize key business metrics such as sales performance, product categories, outlet distribution, and customer ratings. The dashboard enables stakeholders to explore the data dynamically and make informed business decisions.

---

## 🎯 Business Problem

Quick commerce platforms like Blinkit operate across multiple outlets and cities, generating large volumes of transactional data. Without proper analysis, it becomes difficult to identify sales patterns, evaluate outlet performance, and understand customer demand.

This project aims to analyze sales data and answer key business questions such as:

* Which product categories generate the highest sales?
* How do outlet size and location impact revenue?
* What is the overall customer satisfaction level?
* Which outlet types perform best?

The solution is an **interactive Power BI dashboard** that provides both high-level KPIs and detailed analytical insights.
